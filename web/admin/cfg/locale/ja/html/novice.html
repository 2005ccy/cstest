1. 新致クラウドリソースの申請方法

新致クラウドはオープンソースフレーム、 クラウドサーバー、 マネージコード及びクラウドデータベース四つの部分を含みます。プロジェクトを作成した後、"私の"プラットフォームの"マイ・リソース"ページから、需要なクラウドリソースを申請することができます。また、プロジェクトの中で、項目リソースから申請することもできます。申請の場合は、上記四つの部分の“ 立即启用”ボタンをクリックします。ページを入りますと、“ + 申请” ボタンをクリックして設置をします。それから、クラウドリソースを申請することは成功になっています。

2. タスクプロジェクトのリマインドを設定する方法。 そして、 このリマインドを取消した方法。 届くはずのメールが届かない。

新致開発クラウドシステムの右上のユーザアバターをクリックして、アカウント設定 - メッセージの設定を選択します。届けたいメッセージには対応したボタンをオープンします（ボタンは青くなる場合はオープンステータス）。このボタンの変更状況はメールリマインド、デスクリマインド及びメッセージリマインドでお知らせあげます。帯リマインドはオープンした場合、対応する項目の変更状況は携帯でリマインドしてあげます。メッセージのリマインドはある（いくつかの）項目を取消するには、対応するメッセージの設定を閉めます（ボタンは白くなる場合は閉めるステータス）。対応する項目の変更状況は携帯でリマインドしてあげます。メールが届かない場合は、ネットワークが正常であるかどうかを確認してから、メールのスペースが不足することを確認してください。異常なしの場合、アカウントに設定 - メッセージの設定で、対応する項目のメールリマインドはオープンするかどうかをチェックしてください。

3. 開発クラウドはクラウドリソースを提供することができますが、 この利用料のお支払いは？

新致開発クラウドは完全に無料です。弊社のプラットフォームで登録して、正常に起動されている場合、システム内のすべてのリソースを利用することができます。

4. プロジェクトは作成しましたが、 新しいメンバーが需要します、 該当メンバーの招待方法。

新致開発クラウドはメンバーの招待入口が複数を提供します。「私」ページで、右上の新規をクリックして、プルダウンボックスメンバーを招待選択肢を選択します。メンバー招待ポップアップページで項目を選択して、入力ボックスにターゲットメンバーのメールアドレスを入力して、エントリーをクリックします。それから、該当メンバーは選択したプロジェクトに追加しています。或いは、「私のプロジェクト」メニューで、ある具体的なプロジェクトの右上の「」に入ります。描画したプロジェクト情報ページでメンバーリストを選択して、入力ボックスにターゲットメンバーのメールアドレスを入力して、エントリーをクリックします。これもできます。この以外、具体的なプロジェクトをクリックして、ページの最上のタイトルバーのプロジェクトをクリックして、展開したプルダウンボックスにもっとメンバーを招待選択肢を選択します。それから、入力ボックスにターゲットメンバーのメールアドレスを入力して、エントリーをクリックします。

5. クラウドリソースで、 申請したデータベースの使用方法。

クラウドデータベースは申請済みです。バインドしたところで、「コンソール」をクリックすると、アプリケーション環境に該当情報を配置することができます。リンク情報をクリックすると、システムはデータのIP、ポート、ユーザー名、パスワードなど情報が表示されます。アプリケーション環境にこの情報を部署することができます

6. クラウドリソースで、 申請したオープンソースフレームの使用方法。

私たちのオープンソースのフレームワークはLion Frameworkと呼びます。Lion FrameworkはSpring MVC、 Spring、 Hibernate、 Shiro、 Bootstrapの基礎開発フレームワークをベースに、大幅に開発のスピードを向上させます。主な構成要素は：UI： datagrids、 combo、 combotree、 form拡張、 对话框架；対話の枠組み； データの構成要素は：data；一般的なツール：common API serivce dsession 分散セッション redis Redisの読み書きに基づいて、単純な監視モジュールmonitorを操作します。ユーザー名： admin パスワード： 111 aaa。 詳細な使い方については、http/ / lion.newtouchone.com / index.htmlにアクセスしてください。 より詳細な説明があります。

7. クラウドリソースで申請されたマネージコードの使用方法

マネージ コードを有効するには、まず、自分のマネージ ライブラリを作成し、作られたライブラリはリストに表示されます、そのなかの【 申请云版本库】 をクリックして、GitLabにgit倉庫を作成します。

8. どのようにタスクのステータスを変更します

新致開発クラウドは、2 つのタスクのステータスを変更する 方法を提供します、[タスク] パネルにタスクを直接に変更したいステータスの文字にドラッグし、或いはタスク詳細ページに変更したいステータスをクリックして、タスクのステータスが修正されます。

9. プロジェクトまたはタスクを削除したら、 取り戻されますか？

プロジェクトまたはタスクを削除すると、復元できません、プロジェクトに関連するすべてのリソースを削除します。削除機能を使用しないことをお勧めします。システムは、アーカイブ機能を提供し、タスクやプロジェクトは暫く実行しない場合、アーカイブ機能を使用することができます。アーカイブ後、元の内容が完全に保存され、プロジェクト責任者は、いつでもアーカイブの取消ができます。そのため、いつでもデータを復元できますように、私達は削除機能の代わりにアーカイブ機能をお勧めします。

10. パスワードのリセット方法

パスワードを忘れる場合、ログインページに、パスワードの忘れをクリックします。パスワードの設定ページに入ります。あなたのメールアドレスを入力して、パスワードリセットをクリックして、自動にパスワードリセットのメールを送信します。あなたはリンクをクリックして、パスワードをリセットできます。

タスク詳細ページに、このタスク含んだ子タスクを追加できます。
