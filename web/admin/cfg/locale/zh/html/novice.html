1. 如何申请新致云资源？

新致云资源包括开源框架、云服务器、代码托管和云数据库四个部分。用户创建项目后可以从“我的”平台中的“我的资源”页面申请需要的云资源，也可以进入项目中，从项目资源申请。申请时只需要点击四个部分中的“立即启用”按钮，进入页面后点击“+申请”后进行相应的设置，即可申请到您想要的云资源。

2. 如何设置任务项目提醒，如何取消这些提醒？收不到邮件怎么办？

点击新致开发云系统右上方的用户头像，选择账户设置-消息设置，在您想收到消息的对应项后，开启邮件消息提醒（按钮变为蓝色为开启状态），开启消息提醒的相应项的变动情况会通过邮件提醒、桌面提醒和消息提醒来通知到您；如果开启手机提醒，则相应项的变动情况会通过手机提醒到您；如果想取消某一项（几项）的消息提醒，在对应消息设置项后面关闭消息提醒（按钮变为白色为关闭状态），则相应项的变动情况不会提醒到您。如果收不到邮件，请先检查网络是否正常，再查看您的邮箱是否空间不足，如果均无异常，则在账户设置-消息设置中查看对应项的邮件提醒功能是否已开启

3. 新致开发云会提供云资源（服务器，数据库，代码托管等服务），那它怎么收费？

新致开发云是完全免费的，只要您在我们的平台注册并已经成功激活，就可以在我们系统里利用所有资源.

4. 我的项目已经构建成功，我需要新的成员进入，如何邀请成员？

新致开发云提供了多个邀请成员的入口，您可以在“我的”页面，点击页面右上方的新建，选择下拉框中的邀请成员项，弹出邀请成员页面，在此页面先选择项目，然后在输入框中输入目标成员的邮箱，点击回车，该成员成功加入所选项目；也可以进入“我的项目”菜单，进入某个具体的项目卡片右上方的“”，在划出的项目信息页面选择成员列表项，在输入框中输入目标成员的邮箱，点击回车，该成员成功加入所选项目；也可以点击具体项目，点击页面最上面标题栏的项目，在展开的下拉列表中选择邀请更多成员，在输入框中输入目标成员的邮箱，点击回车，该成员成功加入所选项目。

5. 云资源中申请的数据库怎么使用？

云数据库申请成功，在已绑定的服务处，点击“控制台”，您就可以在控制台上执行表操作语句，导入数据等等；点击连接信息，系统会为您展示数据的ip，端口，用户名，密码等信息，您可以把该信息部署到您的应用环境中。

6. 云资源中申请的开源框架如何使用？

我们的开源框架叫做Lion Framework，Lion Framework是一个基于Spring MVC、Spring、Hibernate、Shiro、Bootstrap的基础开发框架，大大提升了开发速度。主要组件有： UI：datagrids、combo、combotree、form扩展、对话框架；数据组件：data；常用工具： common API serivce dsession 分布式会话 redis 基于Redis读写操作 monitor 简单监控模块；用户名：admin 密码：111aaa。详细使用您可以访问http://lion.newtouchone.com/index.html，这里会有更详细的介绍。

7. 云资源中申请的源代码托管如何使用？

启用代码托管可以创建属于自己的代码托管库，列表会显示自己已经创建的代码库，点击【申请云版本库】会在GitLab上创建新的git仓库。

8. 如何更改任务的进度状态变化？

新致开发云提供了两种更改任务进度状态的方式，您可以在任务面板中直接把任务拖动到想要改变的状态下，也可以在任务详情页面上方改变点击任务状态，更改成想要的状态。

9. 项目或任务删除了，还能找回来吗？

我们系统项目或任务删除了，就彻底找不回来了，这个项目所涉及的所有资源也都会被删除。所以建议您最好不要使用删除功能。我们系统提供了归档功能，当您暂时不想进行一个项目、任务，您可以使用归档功能。 归档后，原内容将会被完整保留，项目负责人可以随时取消归档。所以我们建议您用归档功能来代替删除功能，以便可随时恢复内容数据。

10. 密码如何重置？

如果您忘记了密码，在登录页面，点击忘记密码。会进入忘记密码设置页面。输入您的邮箱，点击重置密码，系统会自动发送重置密码的邮件到您的信箱。您点击链接就可以重新设置您的密码。

11. 我想把任务划分为多个子任务，如何去做？

在任务的详情页面，您可以添加该任务包含的子任务。

12. 移动端在哪注册？

目前移动端暂时没有开发注册通道，我们正在优化注册和邀请机制，之后会开放移动端的注册。
